Ackee WP is a simple plugin that easliy adds the JavaScript tracking code required to track vists with Ackee. 

== Description ==

This plugin adds admin settings to easily add the JavaScript tracking code required to track vists in [Ackee](https://ackee.electerious.com/).

According to its author Ackee is:
> Self-hosted, Node.js based analytics tool for those who care about privacy. Ackee runs on your own server, analyses the traffic of your websites and provides useful statistics in a minimal interface.

The plugin requires you to have access to an Ackee Instance installed with your WordPress domain in the Access-Control headers.

Right now the plugin is pretty basic. In the future it may allow advanced features such as tracking based on roles or the ability to use Ackee's advanced analytics.

== Installation ==

The quickest method for installing Ackee WP is:

1. Visit Plugins -> Add New in the WordPress admin
2. Search for "WP Ackee"
1. Click "Install Now"
1. Once the install is complte click "Activate"
1. Go to the Settings -> WP Ackee to add your Ackee settings.

If you would prefer to do things manually then follow these instructions:

1. Upload the `wp-ackee` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the Settings -> WP Ackee to add your Ackee settings.
